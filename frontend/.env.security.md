# API Key 安全使用指南

## ⚠️ 重要提醒

### 开发环境

1. **`.env` 文件已添加到 `.gitignore`**
   - 你的 API Key 不会被提交到 Git
   - 不要手动删除 `.gitignore` 中的规则

2. **`.env.example` 是示例文件**
   - 可以提交到 Git
   - 不要包含真实 API Key

### 生产环境

建议用户通过设置面板配置 API Key：

```typescript
// 在设置页面中添加
await setKimiApiKey("sk-your-api-key");
```

### API Key 获取

Kimi API Key 从 https://platform.moonshot.cn/ 获取：
1. 注册/登录账号
2. 完成实名认证
3. 创建 API Key
4. 充值（有免费额度）

### 费用提醒

- Kimi API 按 token 计费
- moonshot-v1-32k 约 ¥0.012 / 1K tokens
- 导出 3 个对话约 ¥0.05-0.1

## 文件说明

| 文件 | 用途 | 是否提交 Git |
|------|------|--------------|
| `.env` | 真实 API Key | ❌ 否 |
| `.env.example` | 示例配置 | ✅ 是 |
| `src/lib/config/apiKeys.ts` | API Key 读取模块 | ✅ 是 |

## 读取优先级

1. Chrome Storage（用户配置）
2. 环境变量 `.env`（开发配置）
3. 空值（提示用户配置）
