## 1. 产品运行方式

### 1.1 运行模式

**模式A: 浏览器扩展主程序 + 终端适配器（推荐）**

```
┌─────────────────────────────────────────────────────────────────┐
│                        用户工作流程                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  1️⃣  安装VESTI浏览器扩展                                        │
│       └── 自动检测并安装终端适配器                                 │
│                                                                 │
│  2️⃣  日常使用                                                   │
│       ├── 在终端使用Claude Code/Aider编码                        │
│       │   └── 适配器后台自动捕获对话                              │
│       │                                                          │
│       └── 点击浏览器VESTI图标                                    │
│           └── 查看/搜索/管理所有对话                              │
│                                                                 │
│  3️⃣  查看对话                                                   │
│       ├── 📊 概览仪表板（统计/趋势）                              │
│       ├── 💬 对话列表（按项目/时间/工具筛选）                      │
│       ├── 🔍 全局搜索（语义搜索）                                 │
│       └── 📈 代码演进/决策链（Vibe Coding分析）                   │
│                                                                 │
│  4️⃣  导出使用                                                   │
│       ├── 导出为Markdown（含代码块高亮）                          │
│       ├── 导出为JSON（完整数据结构）                              │
│       ├── 生成ADR（架构决策记录）                                 │
│       └── 同步到Obsidian/Notion                                  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**模式B: CLI独立工具（可选）**

```bash
